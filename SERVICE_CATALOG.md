## 🔄 Service Updates

To add new services:

1. **Add to seed.js**:
```javascript
['Service Name', 'Description', price, 'Category', 'duration', 'image_url']
```

2. **Run seed script**:
```bash
cd server && node seed.js
```

3. **Verify in database**:
```bash
sqlite3 database.sqlite
SELECT * FROM services WHERE name = 'Service Name';
```
